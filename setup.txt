1.Создать директорию для проекта

2.В данной директории необходимо создать  виртуальное окружение
 # python3 -m venv venv
 # source venv/bin/activate  #aктивировать окружение

3.В папке проекта необходимо создать файл зависимостей requirements.txt и описать все библиотеки
 # django

4.Необходимо установить все эти зависимости в окружении

 # pip install -r requirements.txt

5.Создаем Django проект
 # django-admin startproject название_проекта .

6.Созданм приложения проекта
 # ./manage.py startapp название

7.Создаем базу данных для проекта
 # psql -> create database название;

8.Настройка проекта
 # Указывем приложения в INSTALLED_APPS
 # Указываем настройки БД
 # Указываем настройки языка, часового пояса

9.Создаем модель пользователя
в settings добавляем настройку AUTH_USER_MODEL

10.Опрелделяем остальные модели

11.Делаем миграции
 # ./manage.py makemigrations
 # ./manage.py migrate

12.Cоздать админа
 # ./manage.py createsuperuser
